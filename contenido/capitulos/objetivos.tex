\chapter{Objetivos}
\label{ch:objetivos}

En este capítulo se definen los objetivos que se persiguen con el desarrollo de este trabajo.

\section{Objetivo general}

El objetivo principal de este \EPStipoTrabajo{} es...

\begin{definicion}[Objetivo general]
Desarrollar un sistema/aplicación/solución que permita... cumpliendo con los requisitos establecidos y aportando valor en el ámbito de...
\end{definicion}

\section{Objetivos específicos}

Para alcanzar el objetivo general, se plantean los siguientes objetivos específicos:

\begin{enumerate}
  \item \textbf{OE1:} Analizar y estudiar el estado actual de...
  
  \item \textbf{OE2:} Diseñar una arquitectura que permita...
  
  \item \textbf{OE3:} Implementar los componentes necesarios para...
  
  \item \textbf{OE4:} Evaluar y validar el sistema desarrollado mediante...
  
  \item \textbf{OE5:} Documentar el proceso y los resultados obtenidos.
\end{enumerate}

\section{Alcance del trabajo}

El alcance de este trabajo incluye:

\begin{itemize}
  \item Desarrollo de la funcionalidad principal
  \item Integración con sistemas existentes
  \item Pruebas y validación del sistema
  \item Documentación técnica y de usuario
\end{itemize}

\textbf{Fuera del alcance:}

\begin{itemize}
  \item Despliegue en producción
  \item Mantenimiento posterior
  \item Funcionalidades adicionales no especificadas
\end{itemize}
